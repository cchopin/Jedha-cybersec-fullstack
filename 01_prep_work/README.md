# 01 - Prepare Your Training (Prep Work)

**Durée :** 1 jour
**Statut :** ✅ Complété

## Objectifs

Module de préparation pour acquérir les bases techniques nécessaires pour la formation en cybersécurité.

## Projets

### bash_training

Collection de scripts Bash pour l'apprentissage de l'automatisation et du scripting système.

#### Fichiers

- **`search_with_bash.sh`**
  Script de recherche de fichiers et répertoires avec options avancées

- **`lizard_toad_snake.sh`**
  Jeu interactif en ligne de commande (pierre-papier-ciseaux version reptile)

#### Utilisation

```bash
# Recherche de fichiers
./bash_training/search_with_bash.sh

# Lancer le jeu
./bash_training/lizard_toad_snake.sh
```

## Compétences acquises

- Scripting Bash
- Manipulation de fichiers et répertoires
- Automatisation de tâches système
- Logique de programmation en shell
