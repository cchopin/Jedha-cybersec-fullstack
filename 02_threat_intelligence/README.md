# 02 - Threat Intelligence

**Durée :** 2 jours
**Statut :** ✅ Complété

## Objectifs

Apprendre à collecter, analyser et partager des informations sur les menaces cybersécurité en utilisant des plateformes de Threat Intelligence.

## Projets

### opencti

Documentation et configuration de la plateforme OpenCTI (Open Cyber Threat Intelligence).

#### Fichiers

- **`opencti_installation_macos.md`**
  Guide complet d'installation d'OpenCTI sur macOS
  - Configuration Docker
  - Installation des dépendances
  - Configuration de l'environnement

- **`opencti_configuration_sources.md`**
  Configuration des sources de données pour OpenCTI
  - Connecteurs de données
  - Flux de menaces (threat feeds)
  - Automatisation de la collecte

#### Installation

Suivre la documentation dans `opencti_installation_macos.md`
