# 03 - Email Security

**Durée :** 1 jour
**Statut :** ✅ Complété

## Objectifs

Comprendre les menaces liées aux emails et apprendre à les analyser et à s'en protéger.

## Sujets étudiés

- Phishing et spear-phishing
- Analyse d'headers d'emails
- SPF, DKIM, DMARC
- Analyse de pièces jointes malveillantes
- Techniques d'ingénierie sociale
- Solutions anti-spam et anti-phishing

## Compétences acquises

- Analyse d'emails suspects
- Identification de tentatives de phishing
- Validation des mécanismes de sécurité email
- Compréhension des protocoles de sécurité email

## Notes

Module complété sans projet Git associé. Compétences pratiques acquises via exercices et analyses de cas réels.
