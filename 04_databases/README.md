# 04 - Databases

**Durée :** 2 jours
**Statut :** ✅ Complété

## Objectifs

Comprendre les bases de données et leurs vulnérabilités du point de vue de la cybersécurité.

## Sujets étudiés

- SQL et bases de données relationnelles
- Injections SQL
- NoSQL et leurs vulnérabilités
- Sécurisation des bases de données
- Chiffrement des données
- Gestion des accès et privilèges
- Backup et recovery

## Compétences acquises

- Compréhension des vulnérabilités SQL injection
- Sécurisation de bases de données
- Requêtes SQL et manipulation de données
- Bonnes pratiques de sécurité des BDD
