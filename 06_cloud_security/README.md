# 06 - Cloud Security

**Durée :** 3 jours
**Statut :** ⏳ À venir

## Objectifs

Apprendre à sécuriser les environnements cloud et comprendre les spécificités de la cybersécurité dans le cloud.

## Sujets à étudier

- AWS, Azure, GCP - Introduction à la sécurité
- IAM (Identity and Access Management)
- Chiffrement des données dans le cloud
- Sécurité des conteneurs et Kubernetes
- Configuration et hardening
- Compliance et gouvernance cloud
- Incident response dans le cloud
