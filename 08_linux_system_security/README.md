# 08 - Linux System Security

**Durée :** 8 jours
**Statut :** ⏳ À venir

## Objectifs

Maîtriser la sécurisation des systèmes Linux, du hardening à la détection d'intrusions.

## Sujets à étudier

- Architecture et permissions Linux
- Hardening de systèmes Linux
- SELinux et AppArmor
- Gestion des logs et monitoring
- Détection d'intrusions sur Linux
- Forensics sur systèmes Linux
- Automatisation avec scripts Bash/Python
- Gestion de patchs et mises à jour

## Projets

Les projets seront ajoutés au fur et à mesure de la progression dans le module.
