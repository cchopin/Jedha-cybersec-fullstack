# 09 - Windows Security

**Durée :** 8 jours
**Statut :** ⏳ À venir

## Objectifs

Apprendre à sécuriser les environnements Windows, d'Active Directory à la détection de malwares.

## Sujets à étudier

- Architecture Windows et Active Directory
- Group Policy Objects (GPO)
- Hardening de systèmes Windows
- PowerShell pour la sécurité
- Analyse de logs Windows (Event Viewer)
- Détection de malwares
- Windows Defender et EDR
- Privilege escalation et mitigation
